Project Delta – Sprint Checklist
Sprint 1 – Core Loop Online
Sprint Goal:
Have a small test map where units can move, gather, return resources, and fight basic enemies with visible HP.

Fix Building Collisions

Make Gatherer return resources to building

Make Unit Have Attack

Make Enemies (simple AI or dummy targets is fine)

Make Health Bars (& HP)

Make Gatherers Fight Back When Attacked !!!

Right Click Context (Enemy = Attack, Resource = Mine, etc.)

Nice to have (if time allows):

Basic Win Condition (e.g., all enemies defeated = Victory)

Sprint 2 – Selection & Control Feel
Sprint Goal:
Make the game feel like an RTS to control: good selection, feedback, and basic UI.

Make Drag select military gatherer specific

Selection Highlight (for units, buildings, and resources)

Change cursor for context (hovered enemy, selectable unit, etc.)

Move Order Marker (brief marker when issuing a move)

UI Panel for selected Unit/Building (name, HP, icon, etc.)

Pop Cap (& Visual Tracker)

Sprint 3 – Combat & Macro Commands
Sprint Goal:
Give the player the tools to manage small armies efficiently.

Attack Move Command ("A" then click)

Military Unit Stances (e.g., Aggressive / Defensive / Hold Position)

Shift Queue Orders (queued move/attack/gather)

Double-Click Select All Same Type

Control Groups (Ctrl+1, etc.)

Idle Worker Hotkey (jump to next idle worker)

Sprint 4 – Camera, Session Flow, First Real Map
Sprint Goal:
Be able to play a full match on a real map from start to finish.

Camera Controls (WASD, Zoom, Pan-Edge-Of-Screen)

Pause & Time Adjust (at least Pause)

Restart level button

Map – First “real” skirmish layout (proto of one Map Idea if you want)

Basic Win/Loss Conditions

Loss Condition (e.g., lose HQ / all buildings)

Win Condition (e.g., destroy enemy base / objective)

Sprint 5 – Audio Feel & Economy Depth Start
Sprint Goal:
Make the game sound alive and start extending the economy.

Select Noise

Build Noise

Gather Start Noise

Background music

Volume Mechanics for Music/SFX

Resource/Income Rate Indicator

Add Secondary "Rare" Resource

Add Third "Craftable" Resource

Later Sprints – Bookmarked Features
These are intentionally left for future sprints once the core is solid.

Upgrades (And An Upgrade Building Or 2)

Defensive Buildings (Walls, Turrets/Attackers)

Production Queue Per Building Display

Make Build Queue

Objective Tracker (for quests, scenario levels)

Minimap

Chatbox (& Cheats)

Help & Tooltips

Visual Assets / Art Upgrade Pass
